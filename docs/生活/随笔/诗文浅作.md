## 1、说明

本文是我的诗词杂文短句小集，记录灵感一闪及咬文嚼字的诗文浅作。<br />

二零二零年九月至二零二二年十月，这两年空闲时间，下班及周末在自学书法，练习毛笔字。前期还算比较认真，后面渐渐半途而废。<br />
虽然书法没练好，但是记录练习书法的方式相当丰富。图片、文章、诗词、博客、社区、公众号、个人网站，每周拍练字照片写人生感想发到网站。文章写了不少，程序员的本职工作也得到了一定程度的锻炼。<br />
《书法练习轨迹--明月几时有》记录本人每周练习书法情况，包含练习图片及一些感悟。<br />
《无为徐生》，本人笔名，表示顺其自然，因时而行。<br />
《徐书法》是徐生的书法，徐某人的书法。<br />
《笔名汉字频率分析》使用java分析本人笔名昵称的分类数量及使用频率。<br />
具体内容可见gitlab的仓库 [calligraphy]( https://gitlab.com/xuyq123/calligraphy ) ，2021到2022年更新较多。 <br />

二零二三及二零二四年沉迷原神，不可自拔。作文《我的原神旅行观测记录》、《纪念一下，三百天打满深渊》等记录在[document]( https://gitlab.com/xuyq123/document )仓库。<br />

二零二五是新的一年，是平凡的一年，也是充满挑战的一年。

## 3、杂句

我的三观：制恒强变真

我的三观：人生观、世界观、价值观，亦即人生信条、五字真言：制 恒 强 变 真。 <br/>
**制**：制怒、自制、克制，不冲动、不贪婪，任何时候都要保持冷静，泰山崩于前而色不变。心平气和才能修身养性。 <br/>
**恒**：持之以恒，学习任何知识、技能都要长久的奋斗，坚持就是胜利。<br/>
**强**：坚强、自强、自信、自立。 <br/>
**变**：变化，见机行事，随机应变。世上唯一的不变就是变化。 <br/>
**真**：真诚、真实。待人以诚、待己以真。认识自己，了解自己的优点缺点，知道自己要干什么。世上乱花迷人眼，需要仔细甄别，方能去伪存真。 <br/>
 
前三条（制、恒、强）是对内，对自己，后两条（变 真）是对外，对别人。只有内外兼修，审慎笃行，两手都要抓，两手都要硬，才能辨别真伪，提高自身。 <br/>
若感觉观点太多，信条很杂，只需要记住一个字：制。即可。 <br/>

在高中的某一天，突然醒悟了以上五字真言，这些年为人处世基本照此原则，不破藩篱。 <br/>
第一条-制怒，实践的比较好，一直都比较冷静平和。 <br/>
后三条-强、变、真，也算是身体力行。 <br/>
第二条-恒 。。。 世间之事，知易行难。行百里者，半途而废者九十。小丑竟是我自己。(⊙︿⊙) <br/>
