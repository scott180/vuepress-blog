# 徐书法

# 一、序章

书法练习轨迹、无为徐生、徐书法，三位一体，循序渐进，成就我的书法练习之路。

# 二、书法练习轨迹

于2020年9月20开始练习书法，且将练习记录及字帖保存在gitee、gitlab、github等代码托管平台，建立项目仓库《书法练习轨迹》 calligraphy (opens new window)。

仓库中主要有四类文件《书法练习轨迹ReadMe》、《书法练习轨迹--明月几时有》、书法练习轨迹图片、其他相关文档。

  • 《书法练习轨迹ReadMe》是自述文件，是git平台的README，作自我介绍之意。包含书法练习轨迹在各个平台的链接。
  • 《书法练习轨迹--明月几时有》记录本人每周练习书法情况：包含练习图片及一些感悟。
  • 书法练习轨迹图片是《书法练习轨迹--明月几时有》文档中图片实际保存地，markdown文件中看到的图片是图片链接。主要有img (opens new window) 和 document (opens new window) 两个仓库。
  • 文档有《古文诗词》、《笔名汉字频率分析》、《无为徐生》、《徐书法》等等我的书法练习相关文档。

# 三、无为徐生

正如2021年8月《无为徐生》诞生所言：

无为徐生是本人昵称，笔名，也是今日头条号、微信公众号、百度百家号的名称。
之所以叫这个名字，是根据本人的籍贯、思想、性别、性格等等综合条件来的。

无为是本人老家，安徽无为，风景秀丽，历史悠久。
无为也是一种道家思想，顺其自然，因时而行。
摒弃妄自作为，远祸慎行，追求朴素节俭、清静寡欲的境界。
徐是本人的姓，男生，所谓斯文有礼谓之生，故名徐生。

综上种种，得名-无为徐生，希冀无为有为，徐生长生。
人生在世，有所作为，文化长生，思想永恒。

# 四、徐书法

《徐书法》是2022年3月注册域名时灵光乍现所想，此名由来如下：

学以致用，见贤而思齐。
之前《书法练习轨迹》都是保存在第三方平台，自己不能随心所欲的记录、更新。
本人是一个程序员，竟然没有属于自己的网站，资料不在自己的服务器，有点说不过去啊。
因此在腾讯云购买了一个轻量应用服务器且注册了新域名：xushufa.cn (opens new window) ，以后这就是我的书法记录平台。
之所以叫：xushufa，中文：徐书法。是截取本人笔名【无为徐生】中的“徐”及书法练习记录【书法练习轨迹】的“书法”组合成【徐书法】。

【徐书法】含义有三：
一、表示是徐生的书法，徐某人的书法。
二、人生如练字，需要不疾不徐，不骄不躁。提醒自己要慢下来，静下去。
三、古有“书圣”“颜体”，高山仰止，今有“徐书法”见贤思齐。十年如一日，徐徐练习，徐书必有法。

# 五、结语

2020年9月创建《书法练习轨迹》；
2021年8月创建《无为徐生》；
2022年3月创建《徐书法》。
三位一体，循序渐进。漫漫长路，持之以恒。
书法练习轨迹是书法练习记录，无为徐生是我的笔名，徐书法则是两者结合。
徐书法是我的书法，是无为徐生的书法，是书法练习轨迹的目标，是我练习书法的梦想。

  • 用自己的方式度过这一生。
  • 人情必有所寄，习惯而成自然。
  • 路漫漫其修远兮，吾将上下而求索。
